Introduction to Artificial Neural Network

Simple way to understand a complex model

Written by Aayush Manandhar on May 22, 2020

Neural networks have been gaining a tremendous interest in the recent decade, although it has been well researched during the mid and late 90s. The main reason for the rise in Neural networks in the recent decade is the improvement in the computing capability that can process vast amounts of data. We can take the example of various applications of Neural networks in the field of facial recognition, Self Driving Cars, Time series analysis, Natural Language processing, and many other tasks a human beings can do.

A neural network is based on the model our brain works. It's basically replicating the learning process of our brain and applying it to the machine. The computer learns to perform specific tasks using neural networks by analyzing various training examples fed into it. The model makes lots of inaccurate predictions in the beginning, but as more and more data gets passed in during training, it can analyze the patterns and make a better projection. It's like teaching a child to distinguish between a horse and a donkey. At first, the child could make some mistake, but as the child sees more pictures and gets experienced enough to differentiate the pattern between a horse and a donkey, the child will successfully know which one is a donkey and which one is a horse.

Traditionally, When working with computers, we have always been giving commands to the computer to execute specific tasks. We can take the example of if-else, switch, and various other logics we give to a program. When we have to build something complicated, the scope is limited, like analyzing vast amounts of time series data, image patterns, and natural language processing. We can't imagine writing a particular code for every new event or circumstance a machine is fed on. For example, if we want to make an image classification for a dog, just by writing, look for a tail, four legs, and other details will not help as every picture we pass to the computer might be different.

We should make the computer able to identify the features of the image and differentiate it from other animals. This is made possible through convolutional neural networks(CNN). Likewise, computers should be able to make a decision based on how it's trained and make a prediction based on their weights and probability. By the use of neural networks, we can train the computer to think and adjust the weight itself rather than depending on a human command.

image source

let's talk about how our human brain works before going deep on the Neural Network. Our human brain has almost 86 billion neurons, and they are connected to each other. A neuron gets its information from dendrites and passes the information or signals to the other end of the cell through axons. But the interesting thing is that the neuron only passes the signal if the total signal received by the cell body exceeds a threshold. The threshold is a point or a barrier that needs to be passed to execute something. The threshold is like a binary digit 0 and 1. It either fires or not. The more neurons in our brain fire, the stronger the connection is in our brain. Canadian psychologist named Donald Hebb researched this theory, where he explained the foundation of how the human brain learns and gets perfect on something. We can also take an example of us learning a foreign language. We are going to make tons of slips at first, but once we practice and understand the basics, it makes the connection between the neurons in our brain stronger and ultimately makes us fluent.

Now, let's head to artificial neural networks. I will be giving a background view of how an artificial neural network works and learn. We have an input layer that takes an input signal and passes it to a hidden layer. A hidden layer transforms the input layer by adjusting the weight and bias. Finally, the output layer or layers, if we are dealing with categorical output, gives the result that's been processed by hidden layers.

Input Layer

Input layers take the input and pass it to the hidden layer. They are independent and aren't changed by other variables we are trying to measure. There is no computation that's done in the input layer, and the only thing it does is pass the information or data to the hidden layer. We can also find a neural network that doesn't have hidden layers, and it's called a single-layer perceptron.

Weights and Bias

Adjusting weights is a crucial process of how a neural network learns and is changed every time a neural network learns. As a neural network keeps learning, it starts analyzing and adjusting the best weight that could lead to a better prediction. Weights can also be called as the strength of the connection as it drastically affects the amount of change in the input will have upon the output. Similarly, a Bias can be compared to the intercept of a linear equation. It helps to give a constant value to every node and shift the activation function to either right or left.

Hidden Layer

In the hidden layer, the neuron takes the weighted sum of all the input values it is getting and applies the activation function. Based on the weighted sum, it determines whether to pass the signal or not. We have various activation functions, but the four popular and mostly used activation functions are threshold activation function, sigmoid function, rectifier function (ReLU), and hyperbolic tangent function. Talking about threshold function, it is a straightforward function that is either 0 or 1. Likewise, The rectifier function is the most commonly used function. This function starts from zero and keeps increasing as the input value progresses. This function is also called a rectified linear unit (ReLU). Similarly, Sigmoid function is a logistic function and has an S-shaped curve and is mostly used in the output layer when we have determined the probability of the output. Finally, the Hyperbolic tangent function is similar to the Sigmoid function, but it ranges from negative 1 to 1. image source

Output Layer

The output layer is the final layer of the artificial neural network and gets input from the hidden layer. An output value can be of three types. It can be continuous, which gives output on a specific range or it could be binary where it's either yes or no, or categorical where we have to select between various categories like distinguishing between a dog, cat, or a fox.

Finally, The important question is, how do weights get adjusted, and how can the best weight be selected? This is where gradient descent and backpropagation come in place, which is a topic for my next article.
